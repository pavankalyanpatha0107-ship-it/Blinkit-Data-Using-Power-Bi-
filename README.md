# 📊 Blinkit Data Analysis Dashboard

## 📌 Overview

This project presents a comprehensive data analysis of Blinkit (grocery delivery) sales and performance using an interactive dashboard. The goal is to extract meaningful insights from the dataset and visualize key business metrics to support data-driven decision-making.

---

## 🚀 Features

* 📈 **Rating Analysis**

  * Rating by Item Type
  * Rating by Outlet Type

* 🏪 **Outlet Insights**

  * Outlet Size Distribution (Small, Medium, High)
  * Outlet Location (Tier 1, Tier 2, Tier 3)
  * Outlet Establishment Trends

* 👀 **Product Analysis**

  * Item Visibility
  * Item Weight by Fat Content

* 📊 **Sales Distribution**

  * Outlet Type Performance Comparison

---

## 🛠️ Tools & Technologies

* **Power BI** – Dashboard creation and visualization
* **Excel / CSV** – Data source
* **Power Query** – Data cleaning & transformation

## 📊 Key Insights

* Supermarket Type 1 outlets generate the highest sales (~0.79M).
* Tier 3 locations show better performance compared to Tier 1 and Tier 2.
* Medium-sized outlets dominate in distribution (~42%).
* Low-fat products have higher item weight distribution.
